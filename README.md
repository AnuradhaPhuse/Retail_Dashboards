# Retail_Dashboards

# Project Title:

Retail Sales Analysis Dashboard using Power BI

# Short Description / Purpose

This project is a professional Power BI dashboard solution designed to analyze online retail sales performance and customer behavior using interactive visualizations.

The project contains two major dashboards:

Online Retail Sales Dashboard – focuses on sales, revenue, products, countries, and shopping trends.
Customer Analysis Dashboard – focuses on customer insights, repeat customers, customer revenue, and customer segmentation.

The dashboards help businesses track KPIs, identify sales trends, understand customer purchasing behavior, and make data-driven business decisions.

The dashboard provides insights into:

Sales performance
Profit analysis
Product category trends
Regional performance
Customer purchasing patterns
Order quantity analysis

# Tech Stack

The dashboard was built using the following tools and technologies:

Power BI Desktop – Main data visualization platform used for report creation
Power Query – Used for data cleaning and transformation
DAX (Data Analysis Expressions) – Used for calculations and data analysis
Microsoft Excel / CSV Dataset – Data source format
Data Modeling – Relationships built between tables for analysis

# Data Source

The dataset used in this project contains retail order and sales information.

Dataset Includes:
Order Details
Customer Information
Product Categories
Sales Amount
Profit
Quantity Ordered
Regional Data
Order Dates

The data was imported into Power BI and cleaned using Power Query Editor.
